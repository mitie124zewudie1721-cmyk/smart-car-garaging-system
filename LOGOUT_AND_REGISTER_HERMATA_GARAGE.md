## 🚨 TROUBLESHOOTING

### Problem: Can't find "Logout" button
**Solution:** Look at top-right corner, next to "System Administrator Admin"

### Problem: Can't find "Register" link
**Solution:** On login page, look below the login button for "Don't have an account? Register"

### Problem: Don't see "GARAGE OWNER" role option
**Solution:** Make sure you're on the registration page, not login page. The role dropdown should have: Car Owner, Garage Owner, Admin

### Problem: Can't find "Add New Garage" button
**Solution:** After login as garage owner, click "My Garages" in left sidebar, then look for "+ Add New Garage" button

### Problem: Form doesn't have 8 sections
**Solution:** 
1. Hard refresh browser: Ctrl+Shift+R
2. Clear cache
3. Make sure backend is running with updated code

### Problem: Can't add multiple services
**Solution:** Look for "+ Add Another Service" button below the first service fields. Click it to add more.
