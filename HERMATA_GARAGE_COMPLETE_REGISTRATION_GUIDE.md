## 🔧 TROUBLESHOOTING

### Problem: Can't login
**Solution:**
1. Check username: `hermata_garage` (no spaces)
2. Check password: `HermataGarage2024!` (case-sensitive)
3. If account doesn't exist, register first

### Problem: Form doesn't load
**Solution:**
1. Hard refresh: Ctrl+Shift+R
2. Clear browser cache
3. Check backend is running: http://localhost:5002/health

### Problem: Can't add services
**Solution:**
1. Click "+ Add Another Service" button
2. Fill in all required fields (name, price, duration)
3. Each service must have a price > 0

### Problem: Can't submit form
**Solution:**
1. Check for red error messages
2. Ensure all required fields filled (marked with *)
3. At least 1 service must be added
4. At least 1 payment method must be selected

### Problem: Submission fails
**Solution:**
1. Check MongoDB connection (see IMPORTANT_FIX_MONGODB_FIRST.md)
2. Check backend console for errors
3. Verify backend is running
