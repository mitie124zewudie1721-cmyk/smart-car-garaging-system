# Garage Registration Form - Complete Guide

## How Each Garage Has Its Own Unique Information

Each garage owner can register their garage with completely different:
- ✅ Services offered
- ✅ Prices for each service
- ✅ Contact information
- ✅ Location
- ✅ Operating hours
- ✅ Payment methods
- ✅ Bank accounts

## How to Fill the Form (Step by Step)

### For CarGarage Hermata:

1. **Login as garage owner**
   - Username: `hermata_garage`
   - Password: `HermataGarage2024!`

2. **Click "Add New Garage"**

3. **Fill Basic Info:**
   ```
   Name: CarGarage Hermata
   Description: Professional car service center...
   ```

4. **Fill Contact:**
   ```
   Phone: +251 911 234567
   Email: hermata.garage@example.com
   ```

5. **Fill Location:**
   ```
   Address: Near Hermata Market, Main Road to Agaro
   City: Jimma
   Region: Oromia
   ```

6. **Add Services (Click "+ Add Service" for each):**
   ```
   Service 1: Car Wash - 300 ETB - 30 min
   Service 2: Oil Change - 800 ETB - 45 min
   Service 3: Tire Service - 500 ETB - 60 min
   Service 4: Brake Service - 1200 ETB - 90 min
   ```

7. **Set Operating Hours:**
   ```
   Mon-Fri: 08:00 - 18:00
   Sat: 08:00 - 17:00
   Sun: 09:00 - 14:00
   ```

8. **Set Capacity:**
   ```
   Total Slots: 8
   ```

9. **Select Payment Methods:**
   ```
   ✓ Cash
   ✓ Telebirr
   ✓ CBE Birr
   ✓ Abysinia Bank
   ```

10. **Add Bank Accounts:**
    ```
    CBE: 1000123456789
    Abysinia: 2000987654321
    ```

11. **Upload License**

12. **Click "Submit for Approval"**

## Key Points

### Each Garage is Unique:
- ✅ Different name
- ✅ Different location
- ✅ Different services
- ✅ Different prices
- ✅ Different contact info
- ✅ Different hours
- ✅ Different payment methods
- ✅ Different bank accounts

### The Form Allows:
- ✅ Add unlimited services
- ✅ Set individual prices for each service
- ✅ Choose which payment methods to accept
- ✅ Set own operating hours
- ✅ Add own bank account details

### After Submission:
- ⏳ Status: "Pending Approval"
- 👨‍💼 Admin reviews the garage
- ✅ Admin approves or rejects
- 📧 Garage owner gets notification
- 🚗 If approved, garage appears in search

## Summary

The form is designed so that:
1. Each garage owner creates their own account
2. Each garage has completely unique information
3. Services and prices are different for each garage
4. Contact info is unique to each garage
5. Payment methods can be different
6. Bank accounts are unique to each garage

This way, customers can compare different garages and choose based on:
- Location (which is closer?)
- Price (which is cheaper?)
- Services (which offers what I need?)
- Hours (which is open when I need?)
- Payment (which accepts my payment method?)
